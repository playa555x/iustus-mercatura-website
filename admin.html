Einrichtung in Azure AD

  1. Gehen Sie zu Azure Portal → App-Registrierungen
  2. Klicken Sie auf "Neue Registrierung"
  3. Name eingeben (z.B. "Website E-Mail")
  4. Nach Erstellung: API-Berechtigungen → Berechtigung hinzufügen
  5. Wählen Sie Microsoft Graph → Anwendungsberechtigungen
  6. Suchen und aktivieren Sie: Mail.Send
  7. Klicken Sie auf "Administratorzustimmung erteilen"
  8. Gehen Sie zu Zertifikate & Geheimnisse → Neuer geheimer Clientschlüssel
